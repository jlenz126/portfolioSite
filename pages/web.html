Portfolio Site

This personal portfolio site started with a simple one page template. I modified the homepage to my liking and instead of having modals open opted to have dedicated pages for each section. The navigation is simple either targeting the section within the page are targeting the needed section on the home page. The connect form as Google reCaptcha v3 imbedded into it and is validated by a PHP script before being emailed to the host server. The site is built using HTML, CSS, Bootstrap, and a little extra JavaScript.
